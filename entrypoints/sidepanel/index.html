Supabase Setup Required

To enable cloud sync and multi-user support, you need to configure Supabase:

  1. Create a Supabase project at supabase.com
  2. Run the migration SQL from supabase/migrations/001_initial_schema.sql
  3. Configure your Supabase URL and Anon Key in the extension

See the README in the supabase folder for detailed instructions.
